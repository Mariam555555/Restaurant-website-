# 💻 Technologies Used
* **HTML5:** For structuring the web pages and forms.
* **CSS3:** For styling, custom animations, flexbox layouts, and modern UI effects (Glassmorphism & Gradients).
* **JavaScript (Vanilla):** For interactive elements such as modal popups and form submission handling.
